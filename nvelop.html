Useage

Dependencies

Nvelop has strong dependencies on JQuery and Bootstrap.

Please download JQuery here and Bootstrap here.

Alternatively, you can include these in the head of your html to reference the latest version of Jquery and Bootstrap

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap.min.css" />

Pages

Organize each page into a div, and give the div a class named "page". Then, assign each page a unique id. This id will be used to show or hide the page. For best results, nest a div with class="container-fluid" inside the page.

<div class="page" id="uniqueId"> Lorem Ipsum </div>

Navigation

For navigation, create a section tag with the id "nav", and put an "a" tag with a class of navbutton, and a href pointing to the id of the page that you want to display

<section id="nav">
<a href="#uniqueId" class="navbutton">One Page</a>
<a href="#anotherUniqueId" class="navbutton">Another Page</a>
</section id="nav">

Extras

Window-height Page

Use the class "fillScreen" on a div to make that div load with a height equal to the window height.

<div class="page fillScreen" id="uniqueId"> Lorem Ipsum </div>

Title Page

Use the class "titleblock" to make a title page that stretches across the width of the screen. This is similar to Bootstrap's "container-fluid" class

<div class="titleblock">Title</div>

About

Building multi-page websites is time consuming. There's either a lot of repeated work, or we have to use tool that are not supported by some web hosts.

Nvelop was built to solve this problem with a simple to use, lightweight package. In a few lines, the structure of a new website is ready to go.

This webpage is built with Nvelop. Visit andrewbfang.com to see another example of Nvelop, and check out more work by the author.
